Para saber más: factores de autenticación


Más allá de crear una contraseña fuerte, GitHub ha estado trabajando para fortalecer la seguridad para el inicio de sesión y el control de versiones del proyecto.

Una de esas medidas es el acceso a través de tokens. En este artículo, Camila Alves explica la importancia y el paso a paso de cómo hacerlo.
