Perfecto. El primer paso que tenemos que hacer es compartir el código. Existen herramientas para ello y una de las principales es la red social Github (Red de compartición de código). La nombro como red social porque al final se comporta como tal. Github no solo es una red para compartir código, sino también para respaldarlo y para trabajar en equipo. Es un conjunto de herramientas muy útiles que nos van a permitir trabajar sobre ellas.

Como Leo comentó, GitHub o Git son herramientas para el versionamiento de código. Entonces, puedo colocar allí diferentes versiones de nuestro sistema que estoy construyendo.
