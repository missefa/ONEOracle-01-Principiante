¿Qué editor de texto usar?
Los siguientes son editores gratuitos y muy usados: Visual Studio Code, Sublime y Atom, puedes escoger cualquiera de ellos, solo refuerzo la importancia de seguir las instrucciones de instalación. Primero, descarga el programa, ejecútalo y luego sigue al asistente hasta el final sin cambiar ninguna configuración.

1 - Visual Studio Code: Es el editor utilizado durante el entrenamiento. Es interesante usarlo para obtener paridad visual durante todo el curso con el instructor, pero no es obligatorio. Para descargarlo, haz clic aquí y en la pantalla de descarga que aparece, elige el sistema operativo que usas.
https://code.visualstudio.com/download

2 - Sublime: Es otra alternativa que puedes usar. Para descargarlo, haz clic aquí y en la pantalla de descarga que aparece, elige el sistema operativo que usas.
https://www.sublimetext.com/3

3 - Atom: Otra alternativa interesante. Puedes descargarlo haciendo clic aquí.
https://atom.io/
