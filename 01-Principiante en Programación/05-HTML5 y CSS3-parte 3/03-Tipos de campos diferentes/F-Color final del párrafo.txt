Color final del párrafo


Considerando el código de abajo:


<p class="parrafo">

p {
    color: blue;
}

p.parrafo {
    color: red;
}

.parrafo {
    color: purple;
}


¿Cuál será el color final de mi párrafo?

a)
Alternativa correta
red

¡Alternativa correcta! El selector p.parrafo es el más fuerte, y el color aplicado es el rojo (red).


b)
purple


c)
blue
